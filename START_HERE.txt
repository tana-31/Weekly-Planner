┌─────────────────────────────────────────────────────────────────┐
│ METHOD 2: MANUAL DEPLOYMENT (Step-by-step control)              │
└─────────────────────────────────────────────────────────────────┘

STEP 1: Create GitHub Account (if you don't have one)
   → Go to https://github.com
   → Click "Sign up"
   → Follow the registration process

STEP 2: Create New Repository
   → Go to https://github.com/new
   → Repository name: weekly-planner
   → Description: A beautiful weekly task planner
   → Keep it PUBLIC ✅ (required for free GitHub Pages)
   → DON'T check any initialization boxes
   → Click "Create repository"

STEP 3: Configure Git & Push Code
   Open Terminal and run:

   cd /Users/I772414/weekly-planner-deploy

   git config user.name "YourGitHubUsername"
   git config user.email "your@email.com"

   git add .
   git commit -m "Initial commit: Weekly Planner v2.0"

   git remote add origin https://github.com/USERNAME/weekly-planner.git
   git branch -M main
   git push -u origin main

   (Replace USERNAME with your actual GitHub username)

STEP 4: Enable GitHub Pages
   → Go to your repository on GitHub
   → Click "Settings" tab (top right)
   → Click "Pages" in left sidebar
   → Under "Source", select "main" branch
   → Click "Save"
   → Wait 1-2 minutes ⏱️

STEP 5: Access Your Live Site! 🎉
   Your planner will be live at:
   https://USERNAME.github.io/weekly-planner/
