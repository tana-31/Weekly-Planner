# 📅 Weekly Planner

A beautiful, intuitive weekly task planner with multi-week support. Plan your entire year with ease!

![Weekly Planner](https://img.shields.io/badge/version-2.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ Features

### 📊 **Multi-Week Planning (52 Weeks)**
- Navigate between weeks with arrow buttons
- Quick jump to any week via dropdown
- Year overview showing all 52 weeks at a glance
- Visual progress tracking for each week

### 📝 **Task Management**
- Create tasks with title, category, effort (hours), priority, and notes
- Drag & drop tasks to assign them to specific days
- **Edit tasks inline** - double-click title or notes to edit
- Color-coded categories (Meetings, Planning, Expo, Development, Other)
- Priority indicators (Normal, High ⚡, Done ✓)

### 📅 **Week View**
- Monday-Friday calendar layout
- Daily hour totals with warning colors (>8h = red)
- Task pool for unassigned tasks
- Actual dates displayed for each day

### 💾 **Data Persistence**
- All data saved in browser localStorage
- Automatic migration from old format
- Works offline - no internet required
- Data persists across sessions

### ⌨️ **Keyboard Shortcuts**
- `←` Previous week
- `→` Next week
- `ESC` Close year view modal
- `Enter` Save title edit
- `Ctrl/Cmd + Enter` Save notes edit

## 📖 How to Use

### Creating Tasks
1. Fill in the task form in the left panel:
   - **Title**: What needs to be done?
   - **Category**: Choose from 5 color-coded categories
   - **Effort**: Hours needed (1-8)
   - **Priority**: Normal, High, or Done
   - **Notes**: Optional details
2. Click "Add Task" - it appears in the Task Pool

### Organizing Your Week
1. **Drag tasks** from the Task Pool to calendar days
2. Each day shows total hours
3. Days turn **orange** at 8h, **red** over 8h

### Editing Tasks
- **Double-click title** to edit task name
- **Double-click notes** to edit or add notes
- **Hover over task** to see edit (✎) and delete (×) buttons

### Navigating Weeks
- **◀ ▶ buttons**: Move week by week
- **Week number**: Click to open dropdown and jump to any week
- **📅 button**: View all 52 weeks in grid layout

### Year Overview
- See all weeks with hours and completion %
- **Blue highlight**: Current viewing week
- **Green background**: 100% completed weeks
- **Dashed border**: Empty weeks
- **Click any week** to jump to it

## 💡 Tips

1. **Weekly Planning**: Start each week by adding tasks to the pool
2. **Time Management**: Keep daily totals under 8 hours
3. **Priorities**: Use ⚡ High for urgent tasks
4. **Completion**: Mark tasks as ✓ Done to track progress
5. **Future Planning**: Navigate ahead to plan upcoming weeks
6. **Review Past Work**: Go back to see what you accomplished

## 🔒 Privacy & Data

- All data stored **locally in your browser**
- No cloud sync, no external servers
- No account required
- No data collection or tracking
- Clear your browser data = clear your tasks

## ⚠️ Important Notes

- Data is **browser-specific** (Safari data ≠ Chrome data)
- Data is **device-specific** (doesn't sync across devices)
- **Backup recommended**: Export feature coming soon!
- Don't clear browser cache/localStorage without backing up

## 🛠️ Technical Details

- **Pure HTML/CSS/JavaScript** - No frameworks
- **localStorage API** for data persistence
- **Responsive design** - Works on desktop, tablet, mobile
- **ISO week numbering** - Week 1 contains January 4th
- **Data format**: JSON structure in localStorage
